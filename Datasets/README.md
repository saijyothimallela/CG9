## Dataset Title

**Tanzil Amharic–English Parallel Corpus (Qur’an Translation Dataset)**

---

## Usage of Dataset

This dataset is used for **Neural Machine Translation (NMT)** tasks, specifically for translating text from **Amharic to English**.
It is suitable for training, validating, and evaluating sequence-to-sequence models such as **RNNs, LSTMs, BiLSTMs, and Transformer-based architectures**.
The dataset can also be used for linguistic analysis and low-resource language research.

---

Tanzil Amharic–English Parallel Corpus

Tanzil Project – Qur’an translations

Religious text (Qur’an verses)

Machine Translation (Amharic → English)

Supervised Learning – Sequence-to-Sequence Translation

CSV

Tanzil Project (Qur’an Translation Dataset)

## Dataset Overview

**Total Records:**
25,197 sentence pairs

**Labeled Records:**
25,197 (fully labeled parallel sentences)

**Classes:**
Not class-based (sequence-to-sequence text pairs)

**Annotation Type:**
Human-translated sentence-level parallel annotations

---

## Why This Dataset?

* Amharic is a **low-resource language**, making this dataset valuable for research.
* High-quality **human-translated parallel corpus** ensures reliable learning.
* Sentence alignment is clean and consistent, suitable for Transformer models.
* Widely used as a **benchmark dataset** for Amharic–English translation tasks.

---

## Features Used

**Feature 1:**
Amharic sentence (source text)

**Feature 2:**
English sentence (target text)

**Feature 3:**
Sentence-level alignment between Amharic and English

---

## Summary

The Tanzil Amharic–English dataset is a high-quality parallel corpus containing over **25k aligned sentence pairs** from Qur’anic translations.
It is well-suited for **low-resource neural machine translation research**, enabling effective training and evaluation of modern deep learning models for Amharic-to-English translation.
